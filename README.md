# Polarity Identifier

  A simple electronic circuit that identifies the positive and negative terminals of a power supply using LEDs. Built as part of Engineering Clinic I at Knowledge Institute of Technology.

  ## Components Used
  - Resistor (1kΩ) x2
  - LED (Red) x1
  - LED (Green) x1
  - Connector pin (1x02) vertical
  - 9V Battery
  - Jumper wires (male to male) x3

  ## How it works
  When connected to a power supply, one LED lights up depending on the direction of current flow — helping identify correct polarity and prevent damage to circuits.
